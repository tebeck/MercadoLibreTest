# MercadoLibre Front End Test

El proyecto se basa en un armado de tres vistas utilizando API de MercadoLibre.

## Getting Started
```
git clone https://github.com/tebeck/MercadoLibreTest.git
```
### Prerequisites

```
NPM
NODEJS
EXPRESS
```

### Installing

First step

```
npm install
```
Second step

```
npm start
```

## Built With

Front End:
- JADE
- CSS3 ( Bootstrap + Own Stylesheet )

Back End:
- NODEJS v8.9.1
- EXPRESS
